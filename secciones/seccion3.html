PRINCIPIOS DE SEGURIDAD DE LA INFORMACIÓN

Accesibilidad

Accesibilidad se refiere a que la información debe estar disponible y accesible para los usuarios autorizados cuando sea necesario. Esto implica implementar medidas como redundancias de sistemas, tolerancia a fallos, y copias de seguridad para asegurar que los sistemas continúen operativos incluso ante fallos o desastres.

Confidencialidad

La confidencialidad asegura que la información solo esté disponible para las personas autorizadas, protegiéndola contra accesos no autorizados. Esto se logra a través de mecanismos como cifrado, autenticación y control de acceso. Es esencial en sectores como el financiero o la salud.

Disponibilidad

Disponibilidad garantiza que los sistemas de información y datos estén siempre accesibles para los usuarios autorizados. Los métodos incluyen el uso de backups, balanceo de carga y planes de recuperación ante desastres para evitar interrupciones.

Autenticación

La autenticación asegura que los usuarios sean quienes dicen ser. Utiliza mecanismos como contraseñas, autenticación multifactor y certificados digitales para verificar la identidad del usuario antes de otorgar acceso a los recursos.

Integridad

Integridad asegura que los datos no sean alterados de manera no autorizada o accidental. Se utilizan técnicas como el uso de firmas digitales y funciones hash para verificar la exactitud de los datos y protegerlos contra modificaciones no autorizadas.

Control de acceso

El control de acceso establece quién tiene permiso para acceder a qué información o recursos, y bajo qué condiciones. Esto puede implicar permisos basados en roles, listas de control de acceso, o restricciones de usuarios según sus privilegios dentro de un sistema.
